**Scenario Description:** This rule detects whether 5G Diameter packets contain payload other than NGAP or DIAMETER.

**Protocol(s):** diameter, sctp_data, ip

**XML Property:**
```xml
<property value="THEN" delay_units="ms" delay_min="0" delay_max="0" property_id="97" type_property="ATTACK" 
    description="5G Diameter packet with wrong SCTP Protocol Identifier">
    <event value="COMPUTE" event_id="1" 
        description="Diameter packet with payload"
           boolean_expression="(diameter.packet_count != 0) &amp;&amp; ((ip.src != ip.dst) )"/>
           
   <event value="COMPUTE" event_id="2" 
      description="SCTP Protocol Identifier different from NGAP (60) or DIAMETER (46)"
         boolean_expression="((sctp_data.data_ppid != 60) &amp;&amp; (sctp_data.data_ppid != 46))"/>
  
</property>
```
This XML property matches the scenario description and adheres to the diameter, sctp_data, ip protocol context. It checks for diameter packets with payload and then verifies that the SCTP Protocol Identifier is either NGAP (60) or DIAMETER (46). If the identifier is different, it triggers an attack recognition event.
